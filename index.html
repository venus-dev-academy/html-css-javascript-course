📌 تگ‌های عنوان (Heading Tags)

تگ‌های h1 تا h6 برای نوشتن عنوان‌ها در صفحات HTML استفاده می‌شن. h1 بزرگ‌ترین و h6 کوچک‌ترین عنوان است.

این یک h1 است - عنوان اصلی

این یک h2 است - زیرعنوان

این یک h3 است - تیتر بخش

این یک h4 است - تیتر فرعی

این یک h5 است - عنوان کوچک
این یک h6 است - کوچک‌ترین عنوان

📌 تگ <p> - پاراگراف

برای نوشتن متن‌های معمولی و پاراگراف‌ها از تگ p استفاده می‌کنیم.

این یک پاراگراف نمونه است که داخل تگ <p> نوشته شده. ما از این تگ برای نوشتن متن‌های طولانی استفاده می‌کنیم.

📌 تگ <div> - بخش‌بندی

تگ div یک ظرف عمومی است. معمولاً برای گروه‌بندی محتوا و استفاده از CSS روی چند بخش با هم به کار می‌ره.

من داخل یک تگ div هستم که بک‌گراند و پدینگ دارد.

📌 تگ <span> - درون‌خطی

تگ span برای گرفتن یک قسمت کوچک از متن به‌کار می‌ره. مثلاً برای رنگ دادن به یک کلمه یا استایل خاص.

سلام دوست خوبم خوش آمدی!

📌 تگ <table> - جدول

برای نمایش داده‌ها به‌صورت جدولی از تگ table استفاده می‌کنیم. هر جدول شامل ردیف‌ها (tr) و سلول‌ها (td / th) است.
